# Cursos e Áreas

## Objetivo
Este arquivo ajuda o RAi a entender cursos, áreas de interesse e caminhos possíveis.

## Cursos comuns
A plataforma pode atender alunos de áreas como:
- Análise e Desenvolvimento de Sistemas;
- Administração;
- Ciências Contábeis;
- Direito;
- Educação Física;
- Engenharia;
- Pedagogia;
- Psicologia;
- Marketing;
- Gestão Comercial;
- Recursos Humanos;
- Enfermagem;
- Arquitetura;
- Letras;
- Sistemas de Informação.

## Áreas de tecnologia
Para alunos de tecnologia, áreas comuns são:
- Front-end;
- Back-end;
- Full stack;
- Banco de dados;
- Inteligência artificial;
- Ciência de dados;
- Segurança da informação;
- Suporte técnico;
- UX/UI;
- Mobile;
- DevOps.

## Como orientar aluno indeciso
Quando o aluno não sabe a área, RAi deve perguntar:
- Você gosta mais de criar telas ou mexer com lógica?
- Gosta de dados?
- Gosta de resolver problemas técnicos?
- Prefere trabalhar com pessoas, sistemas ou análise?
- Quer estágio rápido ou quer construir base primeiro?

## Exemplo de resposta
"Você ainda não precisa escolher para sempre. Escolha uma área para testar por 30 dias. Faça um projeto simples e veja se gosta."
