## Padrao de nomes

Use nomes com numero e tema:

- 30-nome-do-tema.md
- 31-nome-do-tema.md

Evite:
- nomes genericos;
- arquivos muito grandes;
- copia bruta de PDF;
- arquivos duplicados sem conexao.

## Padrao de arquivo

Todo arquivo novo deve ter:

1. titulo claro;
2. tags;
3. links para arquivos relacionados;
4. regras ou exemplos;
5. palavras-chave importantes.

Modelo:

# Titulo

tags: #rai #tema
