## Projetos para tecnologia

### App de tarefas academicas

Problema:
Alunos se perdem com prazos, trabalhos e provas.

Funcionalidades:
- cadastrar tarefas;
- definir prazo;
- marcar como concluida;
- filtrar por materia;
- mostrar prioridade.

Evolucao:
- login;
- notificacao;
- painel de desempenho;
- integracao com calendario.

### Sistema de eventos academicos

Problema:
Eventos ficam espalhados e alunos perdem oportunidades.

Funcionalidades:
- listar eventos;
- buscar por area;
- favoritar;
- inscricao;
- painel do organizador.

### Recomendador de comunidades

Problema:
Aluno novo nao sabe em qual comunidade entrar.

Funcionalidades:
- perguntar curso, interesse e objetivo;
- recomendar comunidades;
- explicar motivo da recomendacao;
- sugerir primeiro post.

### Portfolio pessoal

Problema:
Aluno precisa mostrar pratica para estagio.

Funcionalidades:
- apresentacao;
- lista de projetos;
- habilidades;
- contato;
- links para GitHub e LinkedIn.

## Projetos com IA

### Chatbot de estudos

Entrada:
- materia;
- duvida;
- nivel do aluno.

Saida:
- explicacao curta;
- exemplo;
- pergunta de revisao.

### Resumidor de materiais

Entrada:
- texto ou topicos.

Saida:
- resumo;
- pontos principais;
- perguntas para revisar.

### Organizador de rotina

Entrada:
- prazos;
- tempo disponivel;
- dificuldade.

Saida:
- plano semanal;
- ordem de prioridade;
- revisao.

## Como o RAi deve usar

Quando o usuario pedir ideia, RAi deve:
1. perguntar curso e objetivo;
2. sugerir 2 ou 3 projetos;
3. indicar o menor primeiro passo;
4. propor escopo minimo;
5. conectar com portfolio ou TCC.
