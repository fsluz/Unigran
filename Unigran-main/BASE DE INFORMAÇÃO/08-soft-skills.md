# Soft Skills e Inteligência Emocional

Além do conhecimento técnico, o mercado de trabalho valoriza imensamente as competências comportamentais, conhecidas como soft skills.

## Competências Essenciais

| Skill | Importância |
| :--- | :--- |
| **Comunicação** | Capacidade de expressar ideias de forma clara e ouvir ativamente. |
| **Trabalho em Equipe** | Colaborar com pessoas de diferentes perfis para atingir um objetivo comum. |
| **Resiliência** | Lidar com desafios e aprender com os erros sem perder a motivação. |
| **Pensamento Crítico** | Analisar situações de forma lógica e propor soluções inovadoras. |

## Como Desenvolver
O RAi incentiva os alunos a participarem de projetos em grupo e eventos da Unigran para praticar essas habilidades em um ambiente seguro e colaborativo.
