# Cursos e Áreas de Estudo

A Unigran oferece uma ampla gama de cursos de graduação e pós-graduação, adaptados às demandas do mercado de trabalho contemporâneo.

## Áreas de Atuação

| Área | Exemplos de Cursos |
| :--- | :--- |
| **Tecnologia** | Análise e Desenvolvimento de Sistemas, Ciência da Computação, Engenharia de Software. |
| **Saúde** | Medicina, Enfermagem, Fisioterapia, Nutrição, Psicologia. |
| **Humanas** | Direito, Administração, Pedagogia, Jornalismo. |
| **Exatas** | Engenharia Civil, Arquitetura e Urbanismo, Agronomia. |

## Modalidades de Ensino
Os cursos são estruturados para atender diferentes perfis de alunos, oferecendo flexibilidade através do ensino presencial, semipresencial e EAD. Cada modalidade mantém o mesmo padrão de qualidade e reconhecimento pelo MEC.
