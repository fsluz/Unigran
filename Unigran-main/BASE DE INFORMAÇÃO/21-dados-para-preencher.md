# Dados para preencher no sistema

Este arquivo e um checklist para o RAi funcionar melhor.

## Dados minimos do usuario

- username;
- nome;
- tipo de usuario: aluno, professor, empresa;
- curso;
- semestre;
- bio;
- interesses;
- objetivo atual;
- dificuldade atual;
- habilidades;
- projetos;
- comunidades;
- posts recentes.

## Dados ideais de aluno

Exemplo:

username: vinicius
tipo: aluno
curso: Analise e Desenvolvimento de Sistemas
semestre: 3
interesses: React, Node, IA, carreira, frontend
objetivo atual: conseguir estagio
dificuldade atual: montar portfolio
habilidades: HTML, CSS, JavaScript, React basico
projetos: Unigran, app de tarefas
comunidades: Tecnologia, Estagios, Inteligencia Artificial

## Como preencher para testar o RAi

Use uma mensagem com contexto completo:

"Sou aluno de ADS, 3 semestre, gosto de React e IA, quero conseguir estagio, mas ainda nao tenho portfolio. Tenho conhecimento basico de HTML, CSS e JavaScript. O que faco nos proximos 30 dias?"

Com essa mensagem, o RAi deve:
- reconhecer objetivo de carreira;
- identificar tecnologia;
- sugerir portfolio;
- sugerir comunidades;
- propor plano de 30 dias;
- perguntar sobre projetos existentes.
