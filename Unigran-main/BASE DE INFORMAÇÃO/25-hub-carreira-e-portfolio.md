## Fluxo carreira

objetivo de estagio -> area desejada -> habilidades -> portfolio -> curriculo -> comunidade -> oportunidade

## Checklist de carreira

- definir area;
- criar curriculo de uma pagina;
- organizar LinkedIn;
- organizar GitHub;
- publicar pelo menos 2 projetos;
- escrever README;
- pedir feedback;
- aplicar para vagas realistas;
- participar de comunidade.

## Resposta recomendada

Quando o usuario quiser estagio, RAi deve:

1. reconhecer o objetivo;
2. perguntar curso e semestre se faltar;
3. avaliar se existe portfolio;
4. sugerir plano de 30 dias;
5. recomendar comunidade e oportunidade.
