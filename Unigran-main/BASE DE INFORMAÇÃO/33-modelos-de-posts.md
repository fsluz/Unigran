## Post para pedir feedback

"Pessoal, estou desenvolvendo um projeto de [tema] para praticar [habilidade]. Ainda esta simples, mas queria feedback sobre [ponto especifico]. Quem puder olhar, agradeco."

## Post para mostrar progresso

"Atualizacao do meu projeto: hoje consegui implementar [funcionalidade]. Aprendi [aprendizado]. Proximo passo: [proximo passo]."

## Post para procurar grupo

"Estou procurando colegas para montar um projeto sobre [tema]. Tenho interesse em [area] e posso ajudar com [habilidade]. Alguem anima conversar?"

## Post para pedir mentoria

"Estou estudando [tema] e queria uma orientacao sobre proximos passos. Meu objetivo e [objetivo]. Algum professor, colega ou profissional poderia me dar uma dica?"

## Post para comunidade de tecnologia

"Sou aluno de [curso] e estou estudando [tecnologia]. Quero criar um projeto para portfolio. Pensei em [ideia]. O que voces acham? Alguma sugestao para comecar pequeno?"

## Post para carreira

"Estou organizando meu curriculo e portfolio para buscar estagio em [area]. Ja tenho [habilidades/projetos]. Quais pontos voces acham mais importantes para melhorar?"

## Como o RAi deve usar

Quando o usuario precisar de visibilidade, networking ou feedback, RAi deve sugerir um modelo de post pronto e adaptar ao contexto.
