## Diagnosticos simples

Se usuario tem objetivo mas nao tem plano:
- montar plano curto.

Se usuario tem plano mas nao executa:
- reduzir tarefa e criar rotina.

Se usuario quer estagio mas nao tem portfolio:
- priorizar projeto publicado.

Se usuario tem projeto mas nao divulga:
- sugerir post, GitHub, README e feedback.

Se usuario nao sabe a area:
- sugerir trilha de exploracao de 7 dias.
