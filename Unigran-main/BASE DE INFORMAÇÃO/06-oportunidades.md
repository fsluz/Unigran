# Oportunidades

## Tipos de oportunidade
A plataforma pode exibir:
- estágio;
- vaga júnior;
- projeto acadêmico;
- monitoria;
- evento;
- palestra;
- desafio;
- bolsa;
- processo seletivo;
- networking com empresas.

## Como RAi deve orientar
RAi deve ajudar o usuário a entender:
- se a oportunidade combina com o perfil;
- o que preparar antes de aplicar;
- como melhorar currículo;
- como escrever mensagem de candidatura;
- como montar portfólio.

## Para estágio
Antes de aplicar, o aluno deve ter:
- currículo atualizado;
- LinkedIn simples;
- GitHub ou portfólio, quando for tecnologia;
- disponibilidade de horário;
- noção básica da área.

## Mensagem de candidatura simples
"Olá, tudo bem? Tenho interesse na oportunidade. Sou aluno de [curso], estou no [semestre] e tenho interesse em [área]. Posso enviar meu currículo?"

## RAi deve evitar
- Prometer aprovação.
- Inventar vaga.
- Dizer que empresa vai responder.
- Usar dados de oportunidade sem fonte interna.

## Quando não existir oportunidade
RAi deve dizer:
"No momento, não encontrei uma oportunidade específica na sua base. Mas posso te ajudar a se preparar e procurar pelo tipo certo."
