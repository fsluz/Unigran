# Projetos Acadêmicos

## Objetivo
Ajudar alunos a criar e divulgar projetos.

## Tipos de projeto
- Projeto de disciplina;
- Projeto integrador;
- TCC;
- Portfólio;
- Projeto de pesquisa;
- Projeto de extensão;
- Projeto para comunidade;
- Projeto para empresa parceira.

## Estrutura de projeto
Todo projeto deve ter:
- nome;
- objetivo;
- problema que resolve;
- público-alvo;
- tecnologias ou métodos usados;
- resultado esperado;
- próximos passos.

## Para projetos de tecnologia
Sugestão de estrutura:
- Tela inicial;
- Cadastro/login;
- Banco de dados;
- Listagem;
- Filtro;
- Edição;
- Exclusão;
- Publicação online.

## Como RAi deve ajudar
RAi pode:
- sugerir tema;
- organizar escopo;
- montar checklist;
- melhorar descrição;
- sugerir próximos passos;
- ajudar a transformar projeto em portfólio.

## Exemplo
"Seu projeto precisa parecer menor. Primeiro faça login, cadastro e uma tela principal. Depois você adiciona notificações e IA."
