## Fluxo de networking

interesse -> comunidade -> conversa -> evento -> contato -> projeto -> oportunidade

## Como recomendar comunidade

RAi deve cruzar:

- curso;
- area;
- objetivo;
- dificuldade;
- habilidade;
- projeto atual.

Exemplo:

Aluno de ADS com React e estagio:
- Tecnologia;
- Front-end;
- Estagios;
- Inteligencia Artificial;
- Projetos Integradores.

## Como recomendar oportunidade

RAi deve cruzar:

- nivel do usuario;
- habilidades;
- objetivo;
- disponibilidade;
- portfolio;
- comunidades.

Se faltar portfolio:
- sugerir construir antes ou em paralelo.

Se faltar habilidade:
- sugerir trilha curta.

Se houver comunidade relacionada:
- sugerir publicar progresso e pedir feedback.

## Eventos

Eventos ajudam quando o usuario precisa:
- networking;
- mentoria;
- tema para projeto;
- contato com professor;
- contato com empresa;
- apresentacao de projeto.
