## Analise e Desenvolvimento de Sistemas

Objetivos comuns:
- conseguir estagio em tecnologia;
- criar portfolio;
- escolher entre frontend, backend, dados ou IA;
- criar projeto academico;
- melhorar GitHub.

Trilha inicial:
1. Logica de programacao.
2. HTML, CSS e JavaScript.
3. Git e GitHub.
4. React ou outra tecnologia frontend.
5. Node ou outra tecnologia backend.
6. Banco de dados.
7. Projeto completo publicado.

Projetos recomendados:
- app de tarefas com login;
- sistema de eventos academicos;
- dashboard de notas e frequencia;
- API de oportunidades;
- chatbot simples de estudos;
- portfolio pessoal.

Comunidades recomendadas:
- Tecnologia;
- Front-end;
- Back-end;
- Inteligencia Artificial;
- Projetos Integradores;
- Estagios.

## Administracao

Objetivos comuns:
- estagio administrativo;
- empreendedorismo;
- projetos de gestao;
- analise de mercado;
- carreira em empresas.

Trilha inicial:
1. Comunicacao profissional.
2. Excel ou planilhas.
3. Analise SWOT.
4. Pesquisa de mercado.
5. Plano de negocio.
6. Apresentacao de resultados.

Projetos recomendados:
- plano de negocio;
- estudo de viabilidade;
- simulacao financeira;
- pesquisa de satisfacao;
- estrategia de marketing.

Comunidades recomendadas:
- Administracao e Negocios;
- Empreendedorismo;
- Carreira;
- Projetos;
- Marketing.

## Pedagogia e licenciaturas

Objetivos comuns:
- preparar aulas;
- criar material didatico;
- organizar estagio;
- desenvolver projeto pedagogico.

Trilha inicial:
1. Planejamento de aula.
2. Didatica.
3. Tecnologias educacionais.
4. Avaliacao.
5. Projeto de intervencao.
6. Relatorio de estagio.

Projetos recomendados:
- plano de aula;
- sequencia didatica;
- material interativo;
- projeto de leitura;
- oficina pedagogica.

## Saude

Objetivos comuns:
- estudar para prova;
- organizar estagio;
- criar campanha educativa;
- participar de projeto de extensao.

Trilha inicial:
1. Fundamentos da area.
2. Rotina de revisao.
3. Leitura tecnica.
4. Projeto de extensao.
5. Comunicacao com publico.
6. Etica e privacidade.

Projetos recomendados:
- campanha de prevencao;
- cartilha educativa;
- levantamento de necessidades;
- acao comunitaria;
- apresentacao academica.

## Como o RAi deve usar

Quando souber o curso, RAi deve:
- conectar o curso com uma trilha;
- sugerir comunidades;
- sugerir projeto compativel;
- adaptar linguagem ao semestre;
- perguntar objetivo e dificuldade se faltarem.
