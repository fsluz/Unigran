# Oportunidades de Carreira

A plataforma Unigran conecta alunos a diversas oportunidades de crescimento profissional, desde estágios iniciais até vagas efetivas em grandes empresas.

## Tipos de Oportunidades

| Tipo | Descrição |
| :--- | :--- |
| **Estágio Curricular** | Obrigatório para a conclusão do curso, focado na prática profissional supervisionada. |
| **Estágio Extracurricular** | Oportunidade opcional para ganhar experiência e remuneração durante a graduação. |
| **Trainee** | Programas para recém-formados com foco em desenvolvimento de liderança. |
| **Vagas Efetivas** | Oportunidades de emprego direto para alunos e egressos. |

## Como se Candidatar
Os alunos devem manter seu perfil atualizado no sistema, incluindo currículo e portfólio. A plataforma utiliza inteligência artificial para recomendar vagas que correspondam às habilidades e interesses de cada estudante.
