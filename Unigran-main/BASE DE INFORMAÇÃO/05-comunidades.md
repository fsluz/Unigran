# Comunidades da Plataforma

## O que são comunidades
Comunidades são espaços onde usuários se reúnem por interesse.

Podem ser sobre:
- curso;
- carreira;
- tecnologia;
- estágio;
- pesquisa;
- projetos;
- eventos;
- temas acadêmicos.

## Exemplos de comunidades
- Tecnologia;
- Estágios;
- Inteligência Artificial;
- Front-end;
- Back-end;
- Direito Acadêmico;
- Administração e Negócios;
- Enfermagem;
- Projetos Integradores;
- TCC;
- Empreendedorismo;
- Inglês para carreira.

## Como RAi deve recomendar comunidade
RAi deve analisar:
- curso do usuário;
- interesses;
- objetivo atual;
- semestre;
- projetos;
- dificuldades.

## Exemplo
Usuário:
"Sou de ADS, gosto de React e quero estágio."

RAi pode recomendar:
- Tecnologia;
- Front-end;
- Estágios;
- Projetos;
- Inteligência Artificial.

## Cuidado
RAi não deve recomendar comunidade inexistente como se fosse confirmada.

Quando não souber se existe, falar:
"Procure uma comunidade ligada a tecnologia, estágio ou projetos. Se não existir, talvez valha criar uma."
