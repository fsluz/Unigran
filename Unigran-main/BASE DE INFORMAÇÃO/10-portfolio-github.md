# Portfólio e Presença Digital

Para alunos de tecnologia, o GitHub e o LinkedIn são as principais vitrines para o mercado de trabalho.

## Organizando o GitHub
- **README.md:** Cada projeto deve ter um arquivo explicando o que é, quais tecnologias foram usadas e como rodar o código.
- **Pinned Repositories:** Destaque seus 3 melhores projetos no seu perfil.
- **Contribuições:** Tente manter uma frequência de commits para mostrar que você está praticando constantemente.

## LinkedIn para Estudantes
Mantenha seu perfil atualizado com seu curso atual, habilidades técnicas e links para seus projetos. Conecte-se com colegas, professores e profissionais da área de interesse. O RAi pode revisar a descrição do seu perfil se você solicitar.
