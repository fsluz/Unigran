## Modelo de vaga de estagio

Titulo:
Estagio em [area]

Empresa:
[nome]

Descricao:
Estamos buscando aluno interessado em [area] para apoiar [atividades principais].

Requisitos:
- cursando [curso ou area];
- conhecimento basico em [habilidades];
- vontade de aprender;
- boa comunicacao.

Diferenciais:
- projeto pessoal;
- participacao em comunidade;
- GitHub ou portfolio;
- cursos complementares.

Formato:
remoto, presencial ou hibrido.

Carga horaria:
[horas]

Contato:
[email ou link]

## Modelo de desafio pratico

Desafio:
Criar uma solucao simples para [problema].

Entrega:
- link do projeto;
- breve explicacao;
- prints ou video curto;
- prazo.

Criterios:
- clareza;
- organizacao;
- solucao funcional;
- capacidade de explicar escolhas.

## Como RAi deve ajudar empresa

RAi deve pedir:
- area;
- nivel esperado;
- requisitos reais;
- formato;
- prazo;
- contato;
- se existe bolsa ou remuneracao.

RAi deve evitar:
- vaga vaga demais;
- promessa exagerada;
- requisito impossivel para estagio.

## Como RAi deve ajudar aluno

RAi deve comparar:
- requisitos da vaga;
- habilidades do aluno;
- projetos existentes;
- lacunas;
- proximos passos.
