# Guia de Carreira e Desenvolvimento

O sucesso profissional começa com um planejamento sólido e a construção de uma base técnica e comportamental forte.

## Desenvolvimento para Alunos de Tecnologia
Para estudantes de áreas como ADS e Engenharia de Software, recomenda-se o foco na construção de um portfólio prático.

### Projetos Recomendados
1. **Aplicação CRUD:** Desenvolver um sistema básico de cadastro (Create, Read, Update, Delete) para demonstrar domínio de lógica e banco de dados.
2. **Sistema de Autenticação:** Implementar login e controle de acesso para mostrar preocupação com segurança.
3. **Consumo de API:** Criar uma aplicação que interaja com serviços externos (ex: clima, mapas, finanças).

## Plano de Ação de 30 Dias
- **Semana 1:** Revisão e organização do currículo e perfil no GitHub.
- **Semana 2:** Desenvolvimento de um projeto prático utilizando tecnologias modernas como React ou Node.js.
- **Semana 3:** Publicação do projeto e compartilhamento de aprendizados na comunidade Unigran.
- **Semana 4:** Início das candidaturas para vagas e busca ativa por feedback.
