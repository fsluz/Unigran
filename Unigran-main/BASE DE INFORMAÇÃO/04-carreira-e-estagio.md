# Carreira e Estágio

## Objetivo
Ajudar alunos a dar os primeiros passos profissionais.

## Para alunos de tecnologia
Alunos interessados em programação devem montar um portfólio com 3 projetos pequenos:

1. Uma aplicação CRUD.
2. Um projeto com autenticação.
3. Um projeto consumindo API.

## Plano de 30 dias
### Semana 1
- Organizar GitHub.
- Criar currículo simples.
- Escolher área principal.
- Fazer perfil no LinkedIn.

### Semana 2
- Criar projeto simples com React, HTML/CSS ou outra tecnologia.
- Publicar no GitHub.
- Escrever descrição clara do projeto.

### Semana 3
- Publicar projeto online.
- Fazer post na plataforma UNIGRAN.
- Pedir feedback para colegas e professores.

### Semana 4
- Aplicar para vagas.
- Entrar em comunidades de estágio.
- Melhorar currículo com feedback.

## Quando o aluno estiver perdido
Pergunte:
- Qual curso você faz?
- Em qual semestre está?
- Qual área mais te chama atenção?
- Você já tem algum projeto?
- Você já fez currículo?

## RAi deve evitar
RAi não deve dizer:
- "Você vai conseguir vaga com certeza."
- "Essa vaga é garantida."
- "Seu currículo está perfeito" sem analisar.

## Resposta ideal
"Você precisa sair do zero visível. Primeiro monte um projeto pequeno. Depois publique. Depois peça feedback. Vaga vem depois de prova prática."
